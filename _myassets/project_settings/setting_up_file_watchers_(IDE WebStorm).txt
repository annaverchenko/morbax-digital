1. Jade
http://stackoverflow.com/questions/15492624/auto-compile-jade-in-webstorm-on-windows/15505281#15505281

These setting will let you output .jade files to .html files, retaining the directories structure.

	Open WebStorm, then go to WebStorm settings (ctrl + alt + S)
	Find: Tools > File Watchers > Jade (Create if there is none)

	Program: C:\Users\Игорь\AppData\Roaming\npm\jade.cmd
	Arguments: $FileName$ --pretty --out $ProjectFileDir$\assets\html\$FileDirPathFromParent(jade)$
	Working directory: $FileDir$
	Output paths to refresh: (leave empty)



2. Sass
https://youtrack.jetbrains.com/issue/WEB-9812#

	Open WebStorm, then go to WebStorm settings (ctrl + alt + S)

	2.1 Find: Languages & Frameworks > Compass SCSS

		Enable Compass support (must be checked)
		Compass executable file: C:\Ruby22\lib\ruby\gems\2.2.0\gems\compass-1.0.3\bin\compass
		Config path: E:\<path/to/project-foler>\config.rb


	2.2 Find: Tools > File Watchers > Compass SCSS (Create if there is none)

		Program:					C:\Ruby22\bin\compass.bat
		Arguments:					compile E:/<path/to/project-folder> $UnixSeparators($FilePath$)$
		Working directory:			E:/<path/to/project-folder>
		Output paths to refresh:	E:/<path/to/project-folder>


	2.3 If does not work check:

		2.3.1 Languages & Frameworks > Compass SCSS > Config path

		2.3.2 Tools > File Watchers > Compass SCSS > Arguments, Working directory, Output paths to refresh

		2.3.3 config.rb
